# DailyNews AI

## Overview

DailyNews AI is a full-stack news platform that allows users to discover news, create their own articles, bookmark content, and explore community-created news.

## Features

* User Authentication (JWT)
* Create and Manage Articles
* Community News Feed
* Bookmark Articles
* News Search
* Category-Based News
* Profile Management
* Responsive Design

## Tech Stack

### Frontend

* React
* React Router
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Future Improvements

* Advanced AI-Powered Summaries
* Personalized News Recommendations
* Likes and Comments
* User Profile Photos
* Community Moderation
* Independent News Platform (without external news APIs)

## Installation

1. Clone the repository
2. Install frontend and backend dependencies
3. Configure environment variables
4. Run both servers
